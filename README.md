# CGPA Calculator
A modern web-based CGPA Calculator that helps students calculate their cumulative grade point average (CGPA) across multiple semesters. The application also includes a Target CGPA Planner that calculates the SGPA required in future semesters to achieve a desired CGPA.
## Features
- Calculate CGPA for up to 8 semesters
- Dynamic semester input generation
- Target CGPA Planner
- Responsive and user-friendly interface
- Built with pure HTML, CSS, and JavaScript
## Technologies Used
- HTML5
- CSS3
- JavaScript
